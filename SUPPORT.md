# Support

This project is maintained in spare time, so response times may vary. Here's how to get help:

## 📖 Read First

- [README.md](README.md) — features, setup, usage, and keyboard shortcuts
- [ARCHITECTURE.md](ARCHITECTURE.md) — detailed technical overview
- [CONTRIBUTING.md](CONTRIBUTING.md) — development setup and coding standards

## ❓ Questions

- Search [existing issues](https://github.com/valmapaura/terminalAi/issues) — someone may have asked the same thing
- Start a [GitHub Discussion](https://github.com/valmapaura/terminalAi/discussions) for general questions

## 🐛 Bug Reports

Open a [bug report issue](https://github.com/valmapaura/terminalAi/issues/new?labels=bug&template=bug_report.md). Please include:

- OS version
- App version
- Steps to reproduce
- Expected vs actual behavior
- Any error output

## 💡 Feature Ideas

Open a [feature request](https://github.com/valmapaura/terminalAi/issues/new?labels=enhancement&template=feature_request.md).

## 🙏 Before You Wait

This project is maintained **when time permits**. If you don't hear back within a couple of weeks, feel free to gently ping the thread. Stale issues and PRs will be auto-closed after a period of inactivity.
